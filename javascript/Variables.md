# 변수
### 변수의 선언(let)
- var : 변수의 중복 선언 가능
- let : 변수의 중복 선언 불가능
- 변수를 다시 선언하면 혼란이 커짐! 

### 변수의 종류
- 문자, 숫자, 불리언, 배열
- 동적 타입이라 데이터의 유형은 따로 지정 x

### 객체
- 프로그래밍에서 실제 사물을 모델링하는 코드 구조
- 상자 (너비, 길이, 높이) / 사람 (이름, 키, 몸무게)

### 상수(const)
- 항상 같은 값의 이름을 지정해야 하지만, 상수가 지정하는 값의 내용을 변경할 수 있음.
